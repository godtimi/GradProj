## 配置说明

1. OSS配置
   - 复制示例配置文件：`cp back-end/yymall-api/oss-web/config-example.yaml back-end/yymall-api/oss-web/config.yaml`
   - 修改 `config.yaml` 中的配置信息：
     - 填入您的 AccessKey ID 和 Secret
     - 设置正确的 OSS bucket 访问域名
     - 配置回调 URL
     - 设置上传目录前缀

注意：请勿将包含实际密钥的配置文件提交到代码仓库中。
